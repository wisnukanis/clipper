Kebijakan Privasi Ketentuan Layanan Penghapusan Data

Kebijakan Privasi

Terakhir diperbarui: 12 Mei 2026

RazqaBermain Clipper adalah aplikasi otomasi internal untuk menyiapkan, mengelola, dan menerbitkan konten video pendek ke platform yang dihubungkan oleh pengguna, termasuk layanan Meta seperti Facebook Page dan Instagram.

Data yang Kami Proses

Aplikasi dapat memproses data akun dan izin yang diberikan melalui proses login atau authorization platform, seperti user ID, page ID, token akses, status koneksi, metadata video, URL media, jadwal publikasi, dan status workflow.

Cara Kami Menggunakan Data

  • Menghubungkan akun atau halaman yang dipilih pengguna.
  • Mengunggah dan menerbitkan konten yang dipilih atau disetujui pengguna.
  • Menampilkan status proses, histori workflow, dan hasil publikasi di dashboard.
  • Memperbarui token akses jika platform menyediakan mekanisme refresh yang sah.

Penyimpanan dan Keamanan

Token dan konfigurasi sensitif disimpan sebagai environment variables atau repository secrets yang aksesnya dibatasi. Kami tidak menjual, menyewakan, atau membagikan token pengguna kepada pihak ketiga untuk tujuan pemasaran.

Berbagi Data

Data hanya dikirim ke platform terkait saat dibutuhkan untuk menjalankan fungsi aplikasi, misalnya upload video, validasi token, atau membaca status workflow. Layanan infrastruktur seperti hosting, penyimpanan, dan GitHub Actions dapat memproses data teknis untuk menjalankan aplikasi.

Penghapusan Data

Pengguna dapat meminta penghapusan data aplikasi melalui halaman Instruksi Penghapusan Data. Pengguna juga dapat mencabut akses aplikasi dari pengaturan akun Meta/Facebook atau platform terkait.
