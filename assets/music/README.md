# Emsa Podcast Backsound

Isi folder ini adalah backsound original yang dibuat secara prosedural untuk proyek ini, tanpa sample eksternal dan tanpa mengambil lagu viral/copyrighted dari platform lain.

Mapping otomatis ada di `music-map.json`:

- Theme A: artis, seleb, viral, netizen.
- Theme B: inspirasi, curhat, motivasi, hidup.
- Theme C: komedi, hiburan, santai.
- Theme D: drama, konflik, kontroversi.
- Theme E: fakta, edukasi, bisnis.
- Theme F: religi, ceramah, kajian, dakwah.

Gunakan `BACKGROUND_MUSIC_FILE=auto` supaya renderer memilih track berdasarkan tema dan kata kunci clip. Volume bawaan dibuat kecil karena hanya backsound, bukan musik utama.
